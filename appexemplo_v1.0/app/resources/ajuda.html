Visão Geral

FormDin ou Formulário Dinâmico é um Framework php simples para criar sistema web de forma rápida e fácil.

O FormDin 5 é um meta FrameWork, por ser conconstruido sobre o Adinati FrameWork 7.1 do Pablo Dall'Oglio. Portando não é um FrameWork completo, ele tem total depencendia do Adinati FrameWork 7.1.


Link do GitHub https://github.com/bjverde/formDin5
