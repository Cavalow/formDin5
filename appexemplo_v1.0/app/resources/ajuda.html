Visão Geral

FormDin ou Formulário Dinâmico é um Framework PHP para criar sistema web de forma rápida e fácil.

O FormDin 5 é um meta FrameWork, por ser conconstruido sobre o Adinati FrameWork 7.X do Pablo Dall'Oglio. Portando não é um FrameWork completo, ele tem total depencendia do Adinati. É um Framework de transição do FormDin 4 para o Adianti FrameWork, facilitando a migração. É uma abastração das chamadas do FormDin no Adianti


Link do GitHub https://github.com/bjverde/formDin5
